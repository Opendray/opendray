# Sponsor opendray

opendray is Apache-2.0 and built in the open. Sponsorship pays for the time that keeps it that way: roadmap work, the release pipeline, the 10 README translations, the channel adaptors, the local-first memory layer, and the answers under your "how do I" issues.

If your team runs opendray in production, sponsorship is the cleanest way to keep the project healthy and the maintainer focused.

## Where to sponsor

- **Project (preferred)**: [github.com/sponsors/Opendray](https://github.com/sponsors/Opendray). Money flows to the project, sponsor logos appear in the project's surfaces.
- **Maintainer (fallback)**: [github.com/sponsors/navidrast](https://github.com/sponsors/navidrast). Use this if the org route is unavailable to you (some employers can only sponsor individuals).

Both are tax-receipted via GitHub / Stripe.

## What sponsorship funds

- Continued open development of v2 (sessions, memory layer, channel adaptors, integrations API)
- Maintenance of all 10 README translations and the docs/ guides in step with the code
- The release pipeline: cosign keyless signing, SPDX SBOM, atomically verified self-update
- Community support: issue triage, PR reviews, examples, the Discussions board
- v3 generation R&D, kept under the same Apache-2.0 license

## FAQ

**Is sponsorship required to use opendray?**
No. opendray is Apache-2.0, no telemetry, no cloud login, no subscription. Run it forever without paying anyone.

**My company can only sponsor an organization, not a person.**
Use the org account (`Opendray`). GitHub issues a proper invoice via Stripe.

**My company can only sponsor a person, not an organization.**
Use the maintainer account (`navidrast`). Same Stripe invoicing.

**Neither of those works for our procurement.**
Email `navid@opendray.dev` and we'll arrange a direct sponsorship contract.

**Can I sponsor a specific feature?**
At the Founding tier, yes, one scheduled per quarter. Below that we factor sponsor requests into roadmap weighting but don't commit to delivery dates.

**Is sponsorship money used for anything besides opendray?**
No. It funds opendray development and the human time around it (docs, support, releases). It does not fund unrelated side projects.
